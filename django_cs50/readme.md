## Описание структуры приложения

| Файл        | Описание                                                  |
| ----------- | --------------------------------------------------------- |
| manage.py   | Мы его почти не трогаем, он запускает команды приложения. |
| settings.py | Конфигурация, меняем, чтобы добавить фичи<br />           |
| urls.py     | Таблица контента всей страницы                            |
| wshi.py     |                                                           |
| asgi.py     |                                                           |
| **init**py  |                                                           |

## Команды

_все команды начинаются с python manage.py_

| Команда         | Описание                |
| --------------- | ----------------------- |
| runserver       | Запуск сервера          |
| startapp _name_ | Создание django проекта |
